## Overview

This project develops machine learning models to predict daily solar energy production from photovoltaic installations using meteorological and irradiance features. Accurate production forecasts are critical for grid operators managing intermittent renewable supply and for solar asset owners optimizing maintenance schedules and revenue projections.

## Methodology

Two tree-based algorithms form the modeling pipeline: a Decision Tree regressor as a baseline and a Random Forest ensemble as the primary model. Feature engineering transforms raw weather station data into predictive inputs including rolling-average irradiance, temperature-humidity interaction terms, and cyclical encodings for day-of-year and hour-of-day.

Model selection and hyperparameter tuning rely on k-fold cross-validation with five folds stratified by season to prevent data leakage from temporal autocorrelation. Grid search over tree depth, minimum samples per leaf, and number of estimators identifies the best-performing configuration.

## Results

The Random Forest model achieves an R-squared of 0.87 on the held-out test set, substantially outperforming the single Decision Tree baseline (R-squared of 0.72). Feature importance analysis reveals that peak solar irradiance, ambient temperature, and cloud cover fraction are the three most influential predictors, collectively accounting for over 75% of the model's explanatory power.

```python
from sklearn.ensemble import RandomForestRegressor
from sklearn.model_selection import cross_val_score
import pandas as pd

# Load engineered features
df = pd.read_csv("solar_features.csv")
X = df[["irradiance_avg", "temp", "humidity", "cloud_cover",
        "day_of_year_sin", "day_of_year_cos", "hour"]]
y = df["energy_kwh"]

# Random Forest with cross-validation
rf = RandomForestRegressor(
    n_estimators=200,
    max_depth=15,
    min_samples_leaf=5,
    random_state=42
)

scores = cross_val_score(rf, X, y, cv=5, scoring="r2")
print(f"Mean R²: {scores.mean():.2f} (+/- {scores.std():.2f})")

rf.fit(X, y)
print("Feature importances:")
for name, imp in zip(X.columns, rf.feature_importances_):
    print(f"  {name}: {imp:.3f}")
```
